# CMPG313 K-Means Clustering Assignment

This repository contains my CMPG313 (Artificial Intelligence) assignment on K-Means Clustering of a Dense Network Topology with Energy Analysis.

## Files
- `KMeans.py` – Python script performing K-Means clustering on a dense network of 600 nodes.
- Screenshots – Includes 2D network plots, 3D scatter plots, and 3D energy surface plots.

## Description
- The project applies **K-Means clustering** to group nodes based on spatial coordinates.
- Energy levels of nodes are analyzed to identify critical nodes and understand energy distribution.
- Visualizations include:
  - Dense 2D network plot with centroids
  - 3D scatter plot showing node energies
  - Smooth 3D surface plot of energy distribution
